Typography Elements

<p>

This is a regular paragraph with automatic styling. Paragraphs use Roboto font with a 1.6 line-height for optimal readability. The color is set to a medium gray (#4F5B6B) for contrast.

You can have multiple paragraphs and they will maintain proper spacing automatically.

<a>

This paragraph contains a link with automatic styling. Links are brand-colored (#088C96) with underline. Hover over the link to see the brand-500 color (#00DEDE).

<strong> & <b>

This paragraph has bold text using strong tag and bold text using b tag. Both are styled with 600 font-weight and darker color.

<em> & <i>

This paragraph contains emphasized text using em tag and italic text using i tag. Both display as italics.

<code>

Here's some inline code styled with a light gray background, monospace font, and brand color. Perfect for referencing variables or short code snippets in text.

Semantic Layout Elements

<article>

Article Element

This is an article element. It has white background, padding, rounded corners, and a subtle shadow. Perfect for blog posts, news items, or standalone content pieces.

Articles can contain all other semantic and text elements nested inside them.

<aside>

Aside Element (Sidebar)

This is an aside element, typically used for supplementary content like sidebars, related links, or callout boxes.

  • Related article 1
  • Related article 2
  • Related article 3
<section>

Section Element

Sections are used to group related content together. They have automatic bottom margin spacing.

<main>

The <main> element wraps all your primary content. It auto-centers content with max-width: 1280px and includes padding.

<header>

The <header> element appears at the top of pages or sections. You saw it at the top of this page with the gradient background.

<footer>

This is a footer element. It has a top border, smaller text, and muted color. Perfect for copyright info and secondary links.

Quick Reference

Text Elements

  • <h1>-<h6> - Headings (auto-styled)
  • <p> - Paragraphs
  • <a> - Links (brand color)
  • <strong>, <b> - Bold text
  • <em>, <i> - Italic text
  • <code> - Inline code
  • <pre> - Code blocks
  • <blockquote> - Quotes

Layout Elements

  • <main> - Main content wrapper
  • <article> - Content cards
  • <section> - Content sections
  • <aside> - Sidebar/supplementary
  • <nav> - Navigation menus
  • <header> - Page/section header
  • <footer> - Page/section footer
  • <table> - Data tables

Column Classes

  • .columns - Grid container
  • .columns-2 - 2-column grid
  • .columns-3 - 3-column grid
  • .columns-4 - 4-column grid

Layout Classes

  • .layout-sidebar - Content + right sidebar
  • .layout-sidebar-left - Left sidebar + content
  • .container-narrow - 768px max (articles)
  • .container-wide - 1600px max (dashboards)
